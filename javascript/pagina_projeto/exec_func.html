Exercícios — Funções

Conjunto de enunciados para revisão em sala. Peça aos alunos para implementar cada item usando funções em JavaScript.

  1. Contar vogais em uma palavra

    Crie uma função contarVogais(palavra) que receba uma string e retorne o número de vogais (a, e, i, o, u) presentes. Considere maiúsculas e minúsculas.

  2. Filtrar números pares de um array

    Crie uma função filtrarPares(numeros) que receba um array de números e retorne um novo array contendo apenas os números pares.

  3. Capitalizar palavras

    Crie uma função capitalizar(frase) que receba uma frase e retorne a mesma frase com a primeira letra de cada palavra em maiúsculo.

  4. Calcular IMC

    Crie uma função calcularIMC(peso, altura) que retorne um objeto ou string contendo o valor do IMC com duas casas decimais e a classificação (Abaixo do peso, Peso normal, Sobrepeso, Obesidade).

  5. Palíndromo

    Crie uma função ehPalindromo(palavra) que verifique se a palavra é palíndroma. A função deve ignorar diferenças entre maiúsculas e minúsculas.

  6. Somar números de um array

    Crie uma função somarNumeros(numeros) que retorne a soma de todos os elementos numéricos do array.

  7. Função de desconto progressivo

    Crie uma função calcularDesconto(valor, desconto) que aplique a porcentagem de desconto sobre o valor e retorne o valor final. Se o desconto for maior que 50%, retorne a mensagem "Desconto inválido".

Sugestões de uso em sala

  • Peça para os alunos escreverem a solução em function, depois reescrever com arrow function.
  • Combine exercícios: por exemplo, use filtrarPares e depois somarNumeros no mesmo conjunto de dados.
  • Incentive tratamento de erros (parâmetros inválidos) e comentários explicando a lógica.
