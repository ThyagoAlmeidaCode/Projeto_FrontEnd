Exercícios de JavaScript

Estruturas de decisão, repetição e funções para iniciantes

Estruturas de Decisão

Exercício 1: Verificador de Idade (Simples)

Crie um programa que verifica se uma pessoa é maior de idade.

Use uma estrutura if/else para verificar se a idade é maior ou igual a 18.

Exercício 2: Calculadora de IMC (Médio)

Crie um programa que calcula o IMC e classifica o resultado.

IMC = peso / (altura * altura). Use if/else if/else para classificar o resultado.

Estruturas de Repetição

Exercício 3: Contador (Simples)

Crie um programa que conta de um número até outro.

Use um loop for para iterar do número inicial até o número final.

Exercício 4: Tabuada (Médio)

Crie um programa que gera a tabuada de um número.

Use um loop for de 1 a 10 e multiplique cada número pelo valor informado.

Funções

Exercício 5: Calculadora Simples (Simples)

Crie funções para as quatro operações matemáticas básicas.

Crie funções separadas para cada operação e chame a função apropriada baseada no botão clicado.

Exercício 6: Validador de E-mail (Médio)

Crie uma função que valida um endereço de e-mail.

Verifique se o e-mail contém "@" e "." e se estes não estão no início ou fim.
