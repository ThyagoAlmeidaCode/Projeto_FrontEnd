JavaScript: Estruturas de Decisão e Repetição

Estruturas de Decisão

if, else if, else

As estruturas de decisão permitem que seu código tome decisões com base em condições.

if (condição) {
    // código executado se a condição for verdadeira
} else if (outraCondição) {
    // código executado se a outra condição for verdadeira
} else {
    // código executado se nenhuma das condições for verdadeira
}

switch

O switch é útil quando você tem várias condições para verificar.

switch (expressão) {
    case valor1:
        // código executado se expressão === valor1
        break;
    case valor2:
        // código executado se expressão === valor2
        break;
    default:
        // código executado se nenhum caso corresponder
}

Estruturas de Repetição

for

O loop for é usado quando sabemos quantas vezes queremos repetir um bloco de código.

for (inicialização; condição; expressão final) {
    // código a ser repetido
}

while

O loop while repete um bloco de código enquanto uma condição for verdadeira.

while (condição) {
    // código a ser repetido
}

do while

Similar ao while, mas garante que o código seja executado pelo menos uma vez.

do {
    // código a ser repetido
} while (condição);

Exercícios

Exercício Fácil 1: Par ou Ímpar

Crie um programa que verifique se um número é par ou ímpar.

Solução:

function verificarParImpar() {
    let numero = parseInt(document.getElementById('numeroParImpar').value);
    let resultado = document.getElementById('resultadoParImpar');
    
    if (numero % 2 === 0) {
        resultado.innerHTML = `O número ${numero} é PAR.`;
    } else {
        resultado.innerHTML = `O número ${numero} é ÍMPAR.`;
    }
}

Exercício Fácil 2: Calculadora Simples

Crie uma calculadora que some dois números.

Solução:

function somar() {
    let numero1 = parseFloat(document.getElementById('numero1').value);
    let numero2 = parseFloat(document.getElementById('numero2').value);
    let resultado = document.getElementById('resultadoSoma');
    
    let soma = numero1 + numero2;
    resultado.innerHTML = `A soma de ${numero1} e ${numero2} é ${soma}.`;
}

Exercício Médio 1: Tabuada

Crie um programa que mostre a tabuada de um número.

Solução:

function gerarTabuada() {
    let numero = parseInt(document.getElementById('numeroTabuada').value);
    let resultado = document.getElementById('resultadoTabuada');
    let tabuada = '';
    
    for (let i = 1; i <= 10; i++) {
        tabuada += `${numero} x ${i} = ${numero * i}
`;
    }
    
    resultado.innerHTML = tabuada;
}

Exercício Médio 2: Verificador de Números Primos

Crie um programa que verifique se um número é primo.

Solução:

function verificarPrimo() {
    let numero = parseInt(document.getElementById('numeroPrimo').value);
    let resultado = document.getElementById('resultadoPrimo');
    let isPrimo = true;
    
    if (numero < 2) {
        isPrimo = false;
    } else {
        for (let i = 2; i < numero; i++) {
            if (numero % i === 0) {
                isPrimo = false;
                break;
            }
        }
    }
    
    if (isPrimo) {
        resultado.innerHTML = `O número ${numero} é PRIMO.`;
    } else {
        resultado.innerHTML = `O número ${numero} NÃO é primo.`;
    }
}
