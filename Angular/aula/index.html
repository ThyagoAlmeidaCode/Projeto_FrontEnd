Introdução ao Angular para Iniciantes

Um framework poderoso para criar aplicações web modernas

O que é Angular e por que usá-lo?

Angular é um framework desenvolvido pelo Google para criar aplicações web de página única (SPAs) de forma organizada e eficiente.

Exemplo do dia a dia:

Imagine que você está criando um site de e-commerce. Com Angular, você pode:

  • Exibir produtos em uma lista (componente de lista)
  • Criar um carrinho de compras que atualiza automaticamente (data binding)
  • Filtrar produtos por categoria (diretivas)
  • Gerenciar o login do usuário (serviços)

Vantagens do Angular:

  • Organização: Separação clara entre lógica e visualização
  • Produtividade: CLI que acelera o desenvolvimento
  • Manutenção: Código mais fácil de manter e testar
  • Comunidade: Grande comunidade e suporte do Google
// Componente de produto em um e-commerce @Component({ selector: 'app-produto', template: ` <div class="produto"> <h3>{{produto.nome}}</h3> <img [src]="produto.imagem" [alt]="produto.nome"> <p>Preço: {{produto.preco | currency}}</p> <button (click)="adicionarAoCarrinho()">Comprar</button> </div> ` }) export class ProdutoComponent { @Input() produto: Produto; adicionarAoCarrinho() { console.log('Produto adicionado ao carrinho!'); } }

Diferenças entre AngularJS e Angular

AngularJS (versão 1) é o primeiro framework, enquanto Angular (versão 2+) é uma reescrita completa.

AngularJS (Antigo)

  • Usava JavaScript tradicional
  • Diretivas complicadas
  • Performance mais lenta
  • Dificuldade com aplicações grandes
  • Não otimizado para mobile

Angular (Atual)

  • Usa TypeScript (mais moderno)
  • Arquitetura baseada em componentes
  • Alta performance
  • Melhor para aplicações grandes
  • Otimizado para mobile
Importante:

Se você está começando agora, use Angular (versões 2+). AngularJS está descontinuado e não é mais recomendado para novos projetos.

Conceitos Fundamentais do Angular

Components

Blocos de construção da aplicação. Cada componente controla uma parte da tela.

Exemplo: Componente de cabeçalho, produto, carrinho, etc.

Modules

Agrupam componentes, serviços e outros elementos relacionados.

Exemplo: Módulo de autenticação, módulo de produtos, etc.

Directives

Adicionam comportamento especial aos elementos HTML.

Exemplo: *ngFor para repetir elementos, *ngIf para exibir condicionalmente

// Componente de lista de produtos @Component({ selector: 'app-lista-produtos', template: ` <h2>Nossos Produtos</h2> <div class="produtos"> <app-produto *ngFor="let produto of produtos" [produto]="produto"> </app-produto> </div> <div *ngIf="produtos.length === 0"> Nenhum produto encontrado! </div> ` }) export class ListaProdutosComponent { produtos = [ { nome: 'Smartphone', preco: 999.90, imagem: 'phone.jpg' }, { nome: 'Notebook', preco: 2500.00, imagem: 'laptop.jpg' }, { nome: 'Fones de ouvido', preco: 199.90, imagem: 'headphones.jpg' } ]; }

Configuração do Ambiente

Para começar a desenvolver com Angular, você precisa instalar algumas ferramentas:

Instalar Node.js

Baixe e instale a versão LTS do Node.js a partir do site oficial. O Node.js inclui o npm (gerenciador de pacotes).

# Verificar se Node.js foi instalado corretamente node --version npm --version

Instalar Angular CLI

A CLI (Interface de Linha de Comando) do Angular ajuda a criar e gerenciar projetos Angular.

# Instalar Angular CLI globalmente npm install -g @angular/cli

Criar um novo projeto

Use a CLI para criar um novo projeto Angular.

# Criar novo projeto ng new meu-primeiro-projeto # Navegar para a pasta do projeto cd meu-primeiro-projeto # Executar o servidor de desenvolvimento ng serve

Abrir no navegador

Após executar "ng serve", abra seu navegador e acesse http://localhost:4200/

Você verá a aplicação padrão do Angular funcionando!
